# oh-my-zsh-themes-robyrussel.zsh.theme-venv-prompt
Just a simple way to properly display the venv name in the prompt for oh-my-zsh/robbyrussell.

How?

Replace the theme file with the one available in this repo. The file is located at: `~/.oh-my-zsh/themes`.
